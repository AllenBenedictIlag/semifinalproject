Project

MASK Airlines Reservation

MASK Airline Reservation is based on the concept of booking and canceling flight reservations. Here, the system contains no login feature. The users can easily book flights, cancel flight reservations, and check records. It contains different payment options for the reservations. This mini project contains fewer features but essential ones.

About the Project

Talking about the features of Airline Booking, the system asks for the passengerâ€™s information such as name, passport no, email id. After all these procedures, a reservation is done. As reservation proof, the system provides seat numbers. In order to check tickets, the user has to provide seat no. then only the system check for the respective tickets in the database and displays a result. Canceling a flight reservation is easy through the system, the user just has to provide their seat number.

Features of this project

Packages

Packages includes the voucher like ticket for passengers. This help passengers to access their booking easily and Air travel package means the pre-arranged combination of air transport and ground accommodation and /or other tourist services, sold or offered for sale at an inclusive price. Our packages should consist of 2 adults (18 years old and above) and 2 kids (17 years old and low).

Manual Reservation

This manual reservation is created so that you will be the one who will create your booking and you can choose in a variety of destination. Here you won't be having a packages but you have the rights to use any of the given voucher that you can see in number 3.Coupons so that you would have a reduction on your ticket prizes.

Coupons

Flight Coupon means a portion of a passenger ticket or, in case of an electronic ticket, an electronic coupon, where they indicate the corresponding points of destination and arrival between which the coupon is valid for transportation of passenger and baggage. We indicate coupon code so that it will be easy to access, and each code consist of deduction percentage (5%,10%,15%) which you can use as a discount in your flight.
